# Amo da Minha Vida <3

Uma página especial feita com carinho para guardar momentos, listinhas e mini interações.

## Preview

- Historia e listinhas carregadas dinamicamente
- Contador de dias juntos
- Galeria de fotos com layout responsivo sem cortar imagens
- Player do Spotify flutuante com botao de minimizar
- Mini game "Fuja da Powder" com pontuação e recorde

## Funcionalidades

### Pagina principal
- Cards de listas vindos de um Gist publico
- Card fixo "Estamos juntos há..." com contador automatico
- Visual clean preto e branco com destaque no contador
- Player Spotify flutuante responsivo para desktop e mobile

### Mini game
- Movimento suave por tecla pressionada continuamente
- Controles touch para celular
- Dificuldade progressiva
- Pontuação em tempo real
- Recorde salvo no navegador (localStorage)

### Pagina de fotos
- Galeria em estilo masonry (colunas adaptativas)
- Sem corte e sem deformacao das imagens
- Cards com legenda e hover suave
- Totalmente responsiva

## Tecnologias

## Como rodar localmente

1. Clone o repositorio
2. Abra a pasta do projeto no VS Code
3. Execute com Live Server (ou Five Server)
4. Abra no navegador:
   - index.html
   - fotos.html

## Personalização

- Listas dinamicas: altere o Gist usado em index.html
- Playlist: troque o link embed do Spotify
- Fotos: adicione/remova imagens na pasta fotos e ajuste os cards em fotos.html
